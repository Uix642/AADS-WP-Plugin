== Description ==

This plugin allows you to easily integrate https://a-ads.com/ banner advertisement into your website using WordPress widgets.
It's as simple as drag and drop 


== Frequently Asked Questions ==

= Does it support adaptive ad units =

This plugin is capable of using all sizes of A-ADS ad units, including "Adaptive".
